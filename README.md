# Generador de Portadas ADAS

Aplicación web en TypeScript para generar portadas de Actividades de Aprendizaje, prácticas de laboratorio y productos finales de la Preparatoria Siglo XXI.

## Uso local

```bash
npm install
npm run dev
```

## Catálogos editables

Las materias se modifican en `src/data/subjects.json`.

Los maestros se modifican en `src/data/teachers.json`.

## Scripts

```bash
npm run build
npm run preview
```
